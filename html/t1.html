Tema 1 - GIT

¿Qué es GIT?

GIT es un servicio de comunicación y gestión de información entre local y remoto, con el cual puedes llevar un registro de los cambios en un archivo o el proyecto, definiendo diferentes versiones, que pueden recuperarse en cualquier momento.

GIT esta integrado en la mayoria de IDEs, como Visual Studio Code, PhpStorm, WebStorm, NetBeans, Eclipse, etc.

Almacenamiento de GIT

El almacenamiento de las versiones de GIT se realizan a través de un almacenamiento incremental, esto significa que la primera vez tenemos todos los archivos de golpe, y luego solo se van guardando los archivos que han sido modificados o añadidos.
